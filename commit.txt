ig-pulse: SIC-POVM density matrix framework — 2026-06-27

The 33-stream observatory is a d=12 heteroskedastic POVM approximation.
Full SIC geometry requires 144 = 12² Weyl-Heisenberg elements with uniform
pairwise trace overlap Tr(EᵢEⱼ) = 1/156. Current coverage: 15/144 (10.4%).
108 of the 129 missing elements are synthesisable from existing stream products.

density_matrix.py (new):
- reconstruct(): PSD-projected density matrix from stream readings via
  weighted sum of rank-1 projectors, eigenvalue clip, trace normalisation
- fiducial_proximity(): Tr(ρ·Π_fid) — formal B-state score replacing
  threshold arithmetic; baseline 1/d = 0.083 for maximally mixed
- purity(): Tr(ρ²); von_neumann_entropy(): -Tr(ρ log ρ)
- dominant_coherence_pair(): largest off-diagonal ρ element (cross-primitive)
- sic_overlap_spectrum(): Tr(ρ·Π_{p,q}) for all 144 WH displacements
- sic_heteroskedasticity(): variance of overlap spectrum (0 = uniform coverage)
- Fiducial uses balanced superposition (1/√12)[1,...,1]; replace with
  Scott-Grassl d=12 solution (arXiv:0910.5784) for exact SIC geometry
- WH displacement operators X^p Z^q generated from d=12 clock/shift matrices

sic_povm.py (new):
- STREAM_PRIMITIVE_MAP: 33 streams → (primitive, weight) components
- coverage_analysis(): maps each stream to nearest SIC element, identifies
  covered/missing (p,q) pairs
- _sic_element_character(): dominant primitive axes + phase structure of
  each WH-displaced fiducial state
- missing_stream_proposals(): for each missing element: dominant cross-primitive
  pair, proposed physical observable, synthesisable flag
- CROSS_PRIMITIVE_OBSERVABLES: 40+ cross-primitive pairs → physical stream
  proposals (Wikipedia trends, BGP topology, isotope ratios, options gamma, etc.)
- synthesisable_streams(): 108 elements constructible as nonlinear combinations
  of existing readings — no new data sources required

geo_viz.py:
- /api/all now includes "rho" key with all density matrix metrics per cycle
- /api/density_matrix: full report — rho metrics + SIC coverage + top 30
  missing stream proposals with proposed physical observables

geo_map.html:
- Panel shows ⟨ρ|Π_fid⟩, Purity Tr(ρ²), Entropy S(ρ), dominant coherence
- Fiducial proximity turns gold when > 1.5× baseline (approaching fiducial)
- updatePanel() receives rho from /api/all alongside stats
